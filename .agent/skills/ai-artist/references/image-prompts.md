# Image Generation Prompts

> Techniques for Midjourney, DALL-E, Stable Diffusion, Flux.

---

## Prompt Structure

```
[Subject] + [Style] + [Composition] + [Quality] + [Parameters]
```

---

## Subject Examples

| Type | Example |
|------|---------|
| Portrait | "portrait of young woman with silver hair" |
| Product | "floating smartphone mockup" |
| Scene | "cozy coffee shop interior" |
| Abstract | "geometric patterns in motion" |

---

## Style Keywords

| Category | Keywords |
|----------|----------|
| Lighting | golden hour, neon, rim lighting, volumetric |
| Mood | cinematic, ethereal, dramatic, moody |
| Art style | watercolor, oil painting, digital art, 3D render |
| Era | cyberpunk, retro 80s, futuristic, vintage |

---

## Composition

| Element | Keywords |
|---------|----------|
| Angle | close-up, wide shot, bird's eye, dutch angle |
| Focus | shallow depth of field, bokeh, tilt-shift |
| Environment | fog, rain, particles, lens flare |

---

## Quality Boosters

```
8k, ultra detailed, professional, artstation quality,
award-winning, masterpiece, highly detailed
```

---

## Negative Prompts (Stable Diffusion)

```
Negative: blurry, low quality, distorted, bad anatomy,
watermark, signature, text, cropped
```

---

## Weighted Tokens (Stable Diffusion)

```
(keyword:1.3)   → Increase emphasis
(keyword:0.7)   → Decrease emphasis
[keyword]       → De-prioritize
```

**Example:**
```
(cyberpunk:1.3), neon city, (rain:1.2), cinematic
```
