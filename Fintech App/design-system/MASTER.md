## 🔤 Typography

### Font Pairing: Orbitron / Exo 2

| Role | Font | Weight | Usage |
|------|------|--------|-------|
| **Heading** | Orbitron | 600-700 | H1, H2, Logo |
| **Subheading** | Exo 2 | 500-600 | H3, H4, Labels |
| **Body** | Exo 2 | 400 | Paragraphs, content |
| **Numbers** | Orbitron | 400-500 | Prices, stats, data |

### Google Fonts Import

```css
@import url('https://fonts.googleapis.com/css2?family=Exo+2:wght@300;400;500;600;700&family=Orbitron:wght@400;500;600;700&display=swap');
```

### CSS Font Stacks

```css
:root {
  --font-heading: 'Orbitron', sans-serif;
  --font-body: 'Exo 2', sans-serif;
}
```

## ✨ Key Effects

### Glow Effects

```css
/* Subtle text glow for important numbers */
.value-glow {
  text-shadow: 0 0 10px currentColor;
}

/* Card hover glow */
.card-glow:hover {
  box-shadow: 0 0 20px rgba(245, 158, 11, 0.3);
}
```

### Glass Morphism (Dark)

```css
.glass-card {
  background: rgba(30, 41, 59, 0.8);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(51, 65, 85, 0.5);
  border-radius: 16px;
}
```

### Transitions

```css
/* Standard transition */
.transition-standard {
  transition: all 200ms ease-out;
}

/* Number animations */
.number-update {
  transition: color 300ms, transform 150ms;
}
```

## ⛔ Anti-Patterns (Avoid)

| Don't | Do Instead |
|-------|------------|
| ❌ Use emojis as icons (🚀 📈) | ✅ Use Lucide/Heroicons SVGs |
| ❌ Pure white background | ✅ Use dark `#0F172A` |
| ❌ Scale transform on hover | ✅ Use glow/border effects |
| ❌ Random green/red | ✅ Use semantic `success`/`danger` |
| ❌ Generic sans-serif | ✅ Use Orbitron/Exo 2 |
